Tentang saya

Saya merupakan programming web enthusias dimana saya juga mempunyai impian untuk membangun sebuat kursus berbasis online, beserta saya juga ingin untuk mendirikan sekolah yang bergelut di baidang IT.

Saat ini saya sedang menempuh pendidikan di salah satu sekolah swasta di daerah kuningan jawa barat, yaitu SMK PERTIWI KUNINGAN. Pengalaman saya yaitu saya pernah menjuarai JUARA 1 Festival Fakultas Komputer di Universitas Kuningan antar sekolah SMA/SMK SEDERAJAT daerah Kuningan, Cirebon, Brebes, Majalengka (KUNCI BERSAMA).
